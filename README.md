# 💸 가계부 쓰고 가게

### 돈 관리, 더 쉬워질 수 없을까?

> **가계부 쓰고 가계**는 개인의 소비 습관 개선부터  
> **누구나 쉽게 쓰고, 관리할 수 있는 가계부**를 목표로 합니다.

<br>

이제 혼자 고민하지 말고,  
**가볍게 시작해서 함께 지키는 똑똑한 소비 습관**,  
**가계부 쓰고 가게**와 함께 만들어보세요.

<br>

## 2차 MVP 계획
- 공용 가계부 기능 확장 (기능 확장을 고려하여 데이터 구조 설계)
- AI 소비 패턴 분석
- 음성 수입/지출 저장
- 자동 저장
- 월/연도 별 그래프 분석
- 소비 패턴 알림 (주간 리포트)
- 디자인 시스템 적용
- 다국어 지원

<br>
